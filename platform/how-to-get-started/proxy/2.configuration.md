# Proxy Configuration (AppYamlConfig)

## Provider-Level Configuration

Each provider in `common/config/server.dev.yaml` can set `proxy_url`:

```yaml
providers:
  # Explicit proxy URL (string)
  rally:
    base_url: "https://rally1.rallydev.com/slm/webservice/v2.0"
    proxy_url: "http://proxy.rally.local:8080"

  # Explicitly disabled (boolean false)
  gemini_openai:
    base_url: "https://generativelanguage.googleapis.com/v1beta/openai"
    proxy_url: false

  # Inherit global / env vars (null or omitted)
  servicenow:
    base_url: "https://instance.service-now.com"
    proxy_url: null
```

## Global Network Configuration

The global section defines environment-keyed proxy URLs and agent-level proxies:

```yaml
global:
  network:
    default_environment: "dev"

    # Environment-keyed proxy URLs (used by ProviderClientFactory)
    proxy_urls:
      local: ""                             # No proxy locally
      dev: "http://proxy.dev.local:8080"
      qa: "http://proxy.qa.local:8080"
      prod: "http://proxy.prod.local:8080"

    # Agent-level proxy (equivalent to HTTP_PROXY / HTTPS_PROXY env vars)
    agent_proxy:
      http_proxy: "http://proxy.dev.local:8080"
      https_proxy: "http://proxy.dev.local:8080"

    # TLS settings
    ca_bundle: null
    cert: null
    cert_verify: true
```

## Expose / Compute System

The `expose_yaml_config_compute` section marks fields that undergo runtime
overwrite resolution:

```yaml
expose_yaml_config_compute:
  - proxy_url
```

This enables `applyResolvedOverwrites()` to evaluate template expressions in
`proxy_url` at request time — for example:

```yaml
figma:
  overwrite:
    proxy_url: "{{env.FIGMA_PROXY_URL}}"
```

After resolution, `proxy_url` is either a concrete URL string, `false`, or
`null`, following the standard three-state semantics.

## Storage Providers

Storage providers (`storage.s3`, etc.) also support `proxy_url`:

```yaml
storage:
  s3:
    proxy_url: null      # Inherit global or env
    verify_ssl: null
```

## Resolution Priority

When the health check route builds a client:

```
1. Provider-level proxy_url (from AppYamlConfig after overwrite resolution)
   -> string:  use as explicit proxy URL
   -> false:   force direct connection (no proxy, no env fallback)
   -> null:    fall through to step 2

2. Environment variables
   -> HTTPS_PROXY / HTTP_PROXY (if trust_env = true)

3. Global network config
   -> global.network.proxy_urls[environment]
   -> global.network.agent_proxy.*
```
